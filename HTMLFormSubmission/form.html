Form Elements

The HTML form element can contain one or more of the following form elements:

  • input (click here to see all the input types)
  • label
  • select
  • textarea
  • button
  • fieldset
  • legend
  • datalist
  • output
  • option
  • optgroup

Form Attributes

See the below list

  • action
  • target (_blank, _self {default}, _top, _parent)
  • method (get {default} or post)
  • autocomplete (on or off)
  • novalidate (boolean)
  • accept-charset
  • enctype:
    1. application/x-www-form-urlencoded {default}
    2. multipart/form-data
    3. text/plain [Not recommended]
  • rel
